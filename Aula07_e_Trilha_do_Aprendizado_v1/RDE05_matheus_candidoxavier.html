O que são os Hyperlinks

O hyperlink ou link, é uma função dentro de um documento que serve referência para partes do mesmo documento, de outro, para arquivos ou endereço(s) eletrônico(s).

Os links podem ser remotos, internos, locais, com endereço de correio ou arquivos.


Marcação de Hiperlinks

Marcação de hyperlinks é a parte de um link que implantará o conteúdo desejado por referências.


A tag para acionar a função de marcação de link é < a >
A propriedade para referenciar o conteúdo é < href >


Sintaxe



Hyperlinks internos

Direcionam a um local diferente dentro da mesma página e permite acessar parágrafos ou seções diferentes quando o documento é extenso.


Sintaxe



Hiperlinks locais

Direcionam para outras páginas dentro de um mesmo site.



Sintaxe



Hiperlinks em imagens

É possível abrir hiperlinks de outro conteúdo através das próprias imagens "hyperlinkadas".
Envolve a tag de imagem com a tag de referência de hiperlink



Sintaxe



Hyperlinks para endereço eletrônico

Ex:gmail, hotmail, outlook, yahoo, etc.
